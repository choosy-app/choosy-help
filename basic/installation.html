Choosy Help: Installation guide

A step by step guide to installing Choosy.

  1. Download the latest version of Choosy from www.choosyosx.com.
  2. The download is a ZIP file, depending on how your system is set up it may have unzipped itself automatically, if not just unzip it by double clicking on it in the Finder. It will contain a single file called Choosy.prefPane.
  3. Double click on the Choosy.prefPane file, it will open in System Preferences.
  4. Select whether you want to install Choosy for the current user or for all users.
  5. Set up the way you want Choosy to work. See the configuration guide for more information.
