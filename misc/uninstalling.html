Choosy Help: Uninstalling Choosy

To remove Choosy from your system follow these steps:

  1. Open Choosy in System Preferences
  2. Stop the helper application by clicking the “Stop Helper” button
  3. Uncheck the “Enable Choosy for links” and “Enable Choosy for HTML files” boxes (this will disable Choosy and set the browser at the top of your browser list to be the default browser, once you've removed Choosy you can always change the default browser as normal by using Safari's preferences)
  4. Go back to the main System Preferences menu
  5. Control-click on the Choosy icon and select “Remove Choosy Preference Pane” from the menu
