Choosy Help: Shortcuts integration

Choosy integrates with the macOS Shortcuts app, exposing two actions that you can use in your Shortcuts:

  • The open a URL action will open the given URL with Choosy, applying your rules just as if you'd clicked on a link.
  • The prompt to select a browser action will prompt you to select a browser from all the browsers in your list, and will send the given URL to that browser.
