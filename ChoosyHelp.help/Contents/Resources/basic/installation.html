Choosy Help: Installation guide

A step by step guide to installing Choosy.

  1. Download the latest version of Choosy from choosy.app.
  2. The download is a ZIP file, depending on how your system is set up it may have unzipped itself automatically, if not just unzip it by double clicking on it in the Finder. It will contain a single file called Choosy.pkg.
  3. Double click on Choosy.pkg to run the installer. The installer will check for old versions of Choosy on your system and remove them if needed.
  4. Launch Choosy.app from your /Applications folder in Finder, or via your favourite application launcher (Launch Pad, Spotlight, Alfred, etc.)
  5. Set up the way you want Choosy to work. See the configuration guide for more information.
